What is a Geohash?

A geohash like 9q8y encodes any location on Earth as a short string. The map is repeatedly cut in half — first east/west (longitude), then north/south (latitude), alternating each time. Each cut produces one bit: 1 if the location is in the right/upper half, 0 if left/lower. Every 5 bits are converted to a single character using a 32-symbol alphabet. More characters = smaller cell = higher accuracy.
